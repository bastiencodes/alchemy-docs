---
title: 'alchemy_sendGasOptimizedTransaction'
description: 'Submits a bundle of identical transactions, with different gas values.'
---

<Info>Reinforced Transactions needs to be enabled to use `alchemy_sendGasOptimizedTransaction`.</Info>

### Parameters

<ParamField body="params" type="object[]" required>
  <Expandable title="items" defaultOpen>
    <ParamField type="object" body="param" required>
        <Expandable title="properties" defaultOpen>
            <ParamField body="rawTransactions" type="string[]" required>
            Bundle of signed transactions
            </ParamField>
        </Expandable>
    </ParamField>
  </Expandable>

  
</ParamField>

### Response

<ResponseField name="trackingId" type="string">
The id to query the transaction job status - see `alchemy_getGasOptimizedTransactionStatus`.
</ResponseField>

### Code

<CodeGroup>

```bash cURL
curl --location 'https://eth-mainnet.g.alchemy.com/v2/YOUR_KEY' \
--header 'Content-Type: application/json' \
--data '{
    "id": 1,
    "jsonrpc": "2.0",
    "method": "alchemy_sendGasOptimizedTransaction",
    "params": [
        {
            "rawTransactions": ["0x1", "0x2"]
        }
    ]
}'
```

```typescript viem.ts
import { createClient } from 'viem'
import { mainnet } from 'viem/chains'
import { alchemyActions } from 'viem-actions/alchemy'

const alchemyClient = createClient({
  chain: mainnet,
  transport: http('https://eth-mainnet.g.alchemy.com/v2/YOUR_KEY')
}).extend(alchemyActions)

alchemyClient.sendGasOptimizedTransactions(["0x1", "0x2"]).then(console.log) 

```
